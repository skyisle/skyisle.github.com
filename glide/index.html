Glide 특징

사용하기 편리한 API


Glide.with(this)
	.load("http://goo.gl/kfVV6a")
	.into(imageView);

Placeholder, 애니메이션, Transfomation


Glide.with(myFragment)
	.load(url)
	.centerCrop()
	.placeholder(R.drawable.loading_spinner)
	.crossFade()
	.into(myImageView);
